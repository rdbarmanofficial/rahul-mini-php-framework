# Rahul Mini PHP Framework

A lightweight custom MVC framework built in core PHP.

## Features
- MVC architecture
- Custom Router
- Middleware support
- PDO Database wrapper
- CSRF protection
- Admin routing
- Clean folder structure

## Installation
1. Clone repo
2. Set up .env
3. Import sample database
4. Run in Apache/Nginx
